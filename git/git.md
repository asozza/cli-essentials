
# git: basic commands

- clone repository
```
git clone <ssh-address>
```

- pull & push
```
git pull/push
```

- update repo
```git fetch```

- view changes / status
```
git status
```

- see branches
```
git branch
```

- create new branch
```
git checkout -b <new-branch-name>
```

- switch branch
```
git checkout <branch-name>
```
